变量：可以存储任何数据的一种容器
声明变量：
	1.先声明，后赋值
	2.声明与赋值同时进行
	3.先声明多个变量，然后依次进行赋值
	4.多个变量同时声明与赋值
注意事项：
	1.如果重新声明变量，并且重新赋值，那么变量的值为新的值
	2.如果重新声明变量，但不赋值，变量的结果为原来的结果
	3.如果有声明过程看，但是没有赋值过程，那么结果为undefined，未定义
	（有声明过程，没有赋值过程，结果自动赋值为undefined・）
	4.没有var关键字和赋值过程，那么此变量报错
	5.没有声明过程，但是有赋值过程，那么此变量会变成全局变量，不建议使用

变量名的命名规范：
	1.不可以使用数字开头做变量名
	2.必须以字母，下划线（―）或者$开头
	3.变量名区分大小写
	4.驼峰命名法

基本类型			typeof检测结果
	1.数值类型   number	数值
	2.字符串类型 string	字符串
	3.布尔类型   boolean	布尔值
	4.Null	     object	对象
	5.undefined  undefined	未定义
引用数据类型
	1.函数	     function	函数
	2.对象	     object	对象

双引号与单引号包裹起来的内容都为字符串
