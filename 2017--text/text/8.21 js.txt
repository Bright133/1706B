Javascript作用：
	1.数据的验证，用来验证表单的数据
	2.将内容动态的写入到HTML中
	3.对事件作出响应
	4.可以读取HTML中的内容
	5.可以控制浏览器
	6.可以创建cookies
	7.模拟动画

JavaScript：脚本语言 
	不需要搭载服务器就可以使用的语言


语法特点：基于对象，和事件驱动的松散的解释性语言
	1.对象：一切皆对象
	2.事件驱动：通过加载事件才可以执行的
	3.松散的，弱类型的
	4.解释性：有宿主环境（浏览器）执行

JavaScript组成部分：
	1.ECMAscript语法
		数据类型 运算符 流程控制 函数 面向对象
	2.BOM（browser object Model）
		浏览器对象模型
	3.Dom（document object model）
		文档对象模型
JavaScript：
	1.语言松散 自由性 各个浏览器的兼容性

输出工具
	alert();弹窗；会以文本的格式输出
	console.log();输出到控制台
