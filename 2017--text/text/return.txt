return		返回值
	通过return语句给出函数一个返回值
	停止并且跳出当前的函数

	在函数体内可以有对个return，但是只会执行其中的一个return（if else 语句）


	return语句的返回值
	1.可以是任何数据类型
	2.只能有一个返回值
	3.如果函数没有返回值，自动赋值为undefined

作用域：
	指的就是一段代码的作用范围
	1.全局作用域：任何地方可以访问得到的.
		
	2.函数作用域：只有在函数体内可以访问的

	A.函数外层定义的变量称为全局变量
	B.没有定义直接赋值的变量称为全局变量
	C.参数为私有变量（局部变量）
	D.函数体内定义的变量称为局部变量
